# Video-DL
Downloads video and audio at best quality from youtube and saves it as a mp4 file with the thumbnail as a jpg file in the same directory as the script.

# Requirements:
- ffmpeg-python
```
pip3 install ffmpeg-python
```
- pafy
```
pip3 install pafy
```
- wget
```
pip3 install wget
```
